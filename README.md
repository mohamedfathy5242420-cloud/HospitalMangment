# 🏥 Hospital Management System API

## 📌 Overview
The Hospital Management System (HMS) API is a backend application designed to manage hospital operations efficiently.  
It handles patients, doctors, appointments, and medical records, providing a scalable and organized system for healthcare management.

---

## 🚀 Features

### 👨‍⚕️ Doctor Management
- Add / Update / Delete doctors  
- View doctor profiles  
- Assign doctors to departments  

### 🧑‍🤝‍🧑 Patient Management
- Register new patients  
- Update patient information  
- View patient history  

### 📅 Appointment Management
- Book appointments  
- Cancel or reschedule appointments  
- Track appointment status  

### 📋 Medical Records
- Store patient diagnoses  
- Add prescriptions  
- View medical history  

### 🔐 Authentication & Authorization
- Secure API using JWT  
- Role-based access (Admin, Doctor, Patient)  

---

## 🏗️ Architecture

The project follows **Clean Architecture**:

- **API Layer** → Controllers & Endpoints  
- **Application Layer** → Business Logic & Use Cases  
- **Domain Layer** → Entities & Interfaces  
- **Infrastructure Layer** → Database & Repositories  

---

## 🛠️ Technologies Used

- ASP.NET Core Web API  
- Entity Framework Core  
- SQL Server  
- LINQ  
- JWT Authentication  
- Swagger (API Documentation)
